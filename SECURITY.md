# Security Policy

## Supported Versions

Falcor2 is currently in active development, so the supported version should be considered top-of-tree of the main branch.

## Reporting a Vulnerability

To report a known vulnerability in this project, contact:

- ccummings@nvidia.com
- skallweit@nvidia.com
- tdavidovic@nvidia.com

We will follow up on the vulnerability and aim to address any severe issues promptly.
